☁️ เชื่อมต่อ Google Sheets

ทุกคนที่เปิดไฟล์นี้จะบันทึกและดึงข้อมูลจาก Google Sheet เดียวกัน

📋 วิธีตั้งค่า (ทำครั้งแรกครั้งเดียว)
1. เปิด Google Sheets → สร้าง Spreadsheet ใหม่
2. เมนู Extensions → Apps Script
3. วางโค้ดจากไฟล์ CTStock_GAS.gs → กด Save
4. กด Deploy → New deployment → Web app
5. Execute as: Me · Who has access: Anyone
6. คัดลอก Web app URL มาวางด้านล่าง
